
# 3D Portfolio Website

A modern, simple, and futuristic 3D portfolio website built with React, Three.js, and Framer Motion.

## Features

- **3D Hero Section**: Interactive 3D scene with floating tech workspace.
- **Projects Showcase**: 3D flip cards for project details.
- **About Section**: Service cards with hover effects.
- **Contact Form**: Functional format with 3D Earth model.
- **Responsive Design**: Auto-layout for desktop and mobile.
- **Animations**: Smooth transitions using Framer Motion.

## Tech Stack

- **Core**: React, Vite
- **3D**: Three.js, @react-three/fiber, @react-three/drei, maath
- **Styling**: CSS Modules, Framer Motion
- **Icons**: Lucide React

## Setup & Run

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Customization

- **Content**: Edit `src/constants/index.js` to update projects, services, and experiences.
- **Styles**: Modify `src/index.css` for global theme colors.
